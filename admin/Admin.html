Cadastrar Professor

Cria auth user + registro em public.teachers via create_teacher().

O professor receberá acesso imediato com a senha definida abaixo.
Mín. 8 caracteres

Redefinir Senha

Chama reset_teacher_password(email, nova_senha) no banco.

O professor deverá trocar no próximo acesso.

Cadastrar Aluno

Chama create_student(school_id, class_id, name, birth_date). PIN gerado automaticamente = DDMMAAAA.

A matrícula será gerada automaticamente. O PIN será a data de nascimento no formato DDMMAAAA.

Redefinir PIN do Aluno

Chama reset_student_pin(enrollment, new_pin). PIN deve ter 8 dígitos numéricos (DDMMAAAA).

Formato: dia + mês + ano completo. Ex: 15 de abril de 2010 → 15042010

Importar Alunos via CSV

Chama import_students(school_id, class_id, [{name, birth_date}]). Retorna resumo com erros por linha.

Clique ou arraste o arquivo CSV aqui

Colunas obrigatórias: name, birth_date (YYYY-MM-DD)
Exemplo: Maria da Silva,2010-04-15. Erros por linha são retornados sem interromper a importação.

Criar Nova Turma

Adicione uma turma. O código de acesso do aluno (join_code) será gerado automaticamente.

Usado pelo aluno para logar via student_login_by_code().
